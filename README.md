## 功能特性

### 后端 (Rust)

1. **UDP/EtherCAT数据接收**
   - 100ms采样周期的实时数据接收
   - 二进制协议解析（魔术字0xECAT）
   - 高并发处理（40台机床，每台14个传感器）

2. **ClickHouse时序存储**
   - 振动/温度/位移原始数据表
   - 1分钟自动聚合（物化视图）
   - 高效压缩编码（DoubleDelta/Gorilla）
   - TTL自动数据清理

3. **剩余寿命预测 (RUL)**
   - SKF轴承寿命计算模型（ISO 281标准）
   - 振动RMS趋势分析（线性回归斜率）
   - 温度变化率监测
   - 简化LSTM神经网络调整因子
   - 置信度评估

4. **两级告警系统**
   - 一级振动告警：RMS > 7.1mm/s 持续10秒
   - 二级RUL预警：剩余寿命 < 200小时
   - MQTT推送至车间大屏和MES系统
   - 自动生成维护工单

5. **REST API服务**
   - 机床状态查询
   - 传感器历史数据
   - 健康评分排行榜
   - 告警事件查询
   - 月度统计数据

### 前端 (Canvas + Vanilla JS)

1. **主轴剖面图**
   - Canvas绘制真实主轴剖面图
   - 8个振动传感器位置标记
   - 颜色编码：绿(<2.8) / 黄(2.8-7.1) / 红(>7.1)
   - 点击传感器弹出详情面板
   - 实时脉动动画效果

2. **数据可视化图表**
   - 振动时域波形图
   - FFT频谱分析柱状图
   - 3D振动频谱瀑布图（30分钟跨度）
   - RUL剩余寿命趋势图（含预警线）

3. **监控仪表板**
   - 40台机床健康评分排行榜（Top15）
   - 本月故障统计卡片
   - 最新告警列表（实时更新）
   - 机床快速切换选择器

4. **传感器详情弹窗**
   - 当前RMS值和状态
   - 时域波形缩略图
   - 频谱分析缩略图
   - 24小时RMS趋势图

### 数据模拟器 (Python)

- 40台机床并发模拟（多线程）
- 100ms采样周期，符合EtherCAT标准
- 8振动 + 4温度 + 2位移传感器
- 内置4台故障模拟机床（ID: 5, 12, 28, 35）
- 传感器故障因子放大（1.8~2.5倍）
- 性能统计输出（包/秒）

## 传感器配置

| 类型       | 数量 | 说明                          |
|------------|------|-------------------------------|
| 振动传感器 | 8    | 前/中/后轴承座XYZ三向加速度   |
| 温度传感器 | 4    | 前/中/后轴承 + 定子绕组       |
| 位移传感器 | 2    | 轴向位移 + 径向跳动           |

## 告警阈值

| 级别 | 类型     | 阈值条件                          | 动作                     |
|------|----------|-----------------------------------|--------------------------|
| 一级 | 振动告警 | RMS ≥ 7.1mm/s 持续10秒           | MQTT推送 + 声光报警      |
| 二级 | RUL预警  | 剩余寿命 ≤ 200小时               | MQTT推送 + 生成维护工单  |
| 预警 | RUL提醒  | 剩余寿命 ≤ 500小时               | 系统提示 + 建议维护      |
| 预警 | 振动预警 | 2.8 ≤ RMS < 7.1mm/s              | 传感器黄色标记           |

## 快速开始

### 1. 初始化数据库

```bash
clickhouse-client --multiline < clickhouse/init.sql
```

### 2. 启动后端服务

```bash
cd backend
cargo build --release
./target/release/spindle-monitor-backend
```

### 3. 启动数据模拟器

```bash
cd simulator
python ethercat_simulator.py
```

### 4. 启动前端

```bash
cd frontend
python -m http.server 8000
```

然后访问 `http://localhost:8000`
