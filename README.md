## 功能特性

### 1. 数据采集
- 40台五轴数控机床实时监控
- 每台机床14个传感器（8振动 + 4温度 + 2位移）
- 100ms采样频率，模拟EtherCAT UDP协议
- ClickHouse时序数据存储，带TTL自动过期

### 2. 传感器可视化
- **主轴剖面图**：Canvas绘制主轴横截面，标注轴承、电机、刀具位置
- **传感器标记**：圆点标记传感器位置，颜色根据振动烈度变化
  - 🟢 绿色：< 2.8 mm/s（正常）
  - 🟡 黄色：2.8 - 7.1 mm/s（预警）
  - 🔴 红色：> 7.1 mm/s（危险）
- **点击交互**：点击传感器弹出详情面板

### 3. 振动分析
- **时域波形**：近1小时振动波形实时显示
- **频谱图**：FFT频谱分析，识别轴承故障特征频率
- **瀑布图**：多时间片频谱堆叠，观察频率成分演化

### 4. 剩余寿命预测 (RUL)
- **SKF L10寿命模型**：基于载荷和温度的经典轴承寿命计算
- **LSTM神经网络**：模拟深度学习预测（实际项目中可接入真实模型）
- **多特征融合**：振动RMS趋势 + 温度变化率 + 历史退化数据
- **置信度评估**：输出预测结果的置信区间

### 5. 告警系统
- **一级告警**：振动烈度 > 7.1mm/s 持续10秒 → 振动超限告警
- **二级告警**：RUL < 200小时 → 轴承更换预警
- **MQTT推送**：告警实时推送至车间大屏和MES系统
- **维护工单**：RUL < 500小时自动生成预测性维护工单

### 6. 健康管理
- **健康评分**：综合振动、温度、位移、RUL四维度评分
- **排行榜**：40台机床健康状态排名
- **故障统计**：本月各类告警数量统计
- **历史趋势**：7天内传感器数据趋势分析

## 快速开始

### 1. 初始化数据库

```bash
# 安装ClickHouse后执行
clickhouse-client < clickhouse/init.sql
```

### 2. 启动后端服务

```bash
cd backend

# 安装依赖并编译
cargo build --release

# 设置环境变量（可选）
export CLICKHOUSE_URL="http://localhost:8123"
export UDP_PORT=9999
export HTTP_PORT=8080

# 运行
./target/release/spindle-health-monitor
```

### 3. 启动数据模拟器

```bash
cd simulator

# 正常模式
python ethercat_simulator.py

# 异常模式（模拟部分机床故障）
python ethercat_simulator.py --anomaly

# 指定运行时长（5分钟）
python ethercat_simulator.py --duration 5
```

### 4. 访问前端

直接用浏览器打开 `frontend/index.html`

或者启动一个简单的HTTP服务器：

```bash
cd frontend
python -m http.server 3000
# 访问 http://localhost:3000
```

## 传感器配置

每台机床配置14个传感器：

| 编号 | 位置 | 类型 | 单位 |
|------|------|------|------|
| 1 | 前轴承径向X | 振动 | mm/s |
| 2 | 前轴承径向Y | 振动 | mm/s |
| 3 | 前轴承轴向 | 振动 | mm/s |
| 4 | 后轴承径向X | 振动 | mm/s |
| 5 | 后轴承径向Y | 振动 | mm/s |
| 6 | 后轴承轴向 | 振动 | mm/s |
| 7 | 电机端径向 | 振动 | mm/s |
| 8 | 刀具端径向 | 振动 | mm/s |
| 9 | 前轴承座 | 温度 | °C |
| 10 | 后轴承座 | 温度 | °C |
| 11 | 定子绕组 | 温度 | °C |
| 12 | 环境温度 | 温度 | °C |
| 13 | 轴向位移 | 位移 | mm |
| 14 | 径向跳动 | 位移 | mm |

## 技术栈

### 后端
- **Rust**：高性能系统级编程语言
- **Tokio**：异步运行时
- **Axum**：Web框架
- **ClickHouse**：时序数据库
- **rumqttc**：MQTT客户端

### 前端
- **原生Canvas**：高性能图形绘制
- **原生JavaScript**：无框架依赖，加载快
- **CSS3**：深色科技风格UI

### 模拟器
- **Python**：快速开发模拟脚本
- **struct**：二进制UDP数据包构建

## 告警阈值配置

| 参数 | 阈值 | 级别 | 说明 |
|------|------|------|------|
| 振动烈度 | > 7.1 mm/s | 一级 | 持续10秒触发 |
| RUL | < 200 小时 | 二级 | 轴承更换预警 |
| RUL | < 500 小时 | - | 自动生成工单 |
| 温度 | > 85 °C | 预警 | 温度异常 |
| 位移 | > 0.3 mm | 预警 | 位移异常 |

## 注意事项

1. 本项目包含模拟数据，实际部署时需接入真实EtherCAT总线
2. LSTM预测模块为模拟实现，生产环境需训练真实模型
3. MQTT推送需配置实际的Broker地址和认证信息
4. ClickHouse需根据数据量调整分区和TTL策略
